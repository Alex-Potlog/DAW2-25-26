# Sript scss.sh
Crea una plantilla para proyectos html utilizando scss.
## Modo de empleo
Primero hay que mover el script a la carpeta donde quieres crear el proyecto y despue hay que ejecutarlo.
Se puede ejecutar de dos formas:
- La clasica.
```
sh scss.sh
```
- Una mas corta.
```
./scss.sh
```

Por ultimo elimina el script y ya tendras el proyecto listo.
```
rm scss.sh
```
